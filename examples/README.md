# Examples

This folder contains safe, non-sensitive example content for demos, onboarding, and contributor testing.

Files here should avoid private manuscript content.

## Included

- `sample-chapter.md`: a short chapter fragment for review testing

## Suggested Usage

1. Start the app locally
2. Open the review page
3. Paste `sample-chapter.md`
4. Try behavior, dialogue, and writing-style review combinations

# Novel Assistant v0.1.0

First public release of Novel Assistant.

Language notes:

- English README: [README.md](../../README.md)
- Traditional Chinese README: [README.zh-TW.md](../../README.zh-TW.md)
- Traditional Chinese release notes: [v0.1.0.zh-TW.md](v0.1.0.zh-TW.md)

Novel Assistant is a local-first AI writing companion for fiction projects, built with Go and Ollama. This release establishes the core review workflow and the open-source project foundation.

## Highlights

- Character behavior review with streaming output
- Dialogue style review per character
- Writing style review with reusable `data/style/*.md` profiles
- Relationship, timeline, and foreshadow trackers
- Local RAG context display during review
- Markdown export for review reports

## Included in This Release

### Review Workflows

- Behavior consistency review
- Dialogue style review
- Writing style review
- Automatic character detection from chapter text when no character is manually selected

### Story Project Structure

- Markdown-based character profiles
- Markdown-based worldbuilding files
- Markdown-based writing style guides
- Local JSON persistence for relationship, timeline, and foreshadow trackers

### Project Infrastructure

- MIT license
- CI workflow for formatting, tests, and build checks
- Contributing, security, and code of conduct documents
- Issue templates and pull request template
- Example chapter content and architecture documentation

## Notes

- This is an early but usable release intended for local-first workflows.
- Ollama must be running locally with compatible models available.
- Generated local state files under `data/` remain intentionally untracked.

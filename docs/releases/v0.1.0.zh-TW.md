# Novel Assistant v0.1.0

Novel Assistant 的第一個公開版本。

這是一個以本地優先為核心的小說寫作輔助工具，使用 Go 與 Ollama 構建。這個版本建立了專案的核心審查流程，也補齊了開源專案所需的基礎設施。

## 重點功能

- 串流式角色行為一致性審查
- 角色對白風格審查
- 支援 `data/style/*.md` 的寫作風格審查
- 關係圖、時間軸與伏筆追蹤
- 審查時顯示本地 RAG 參考上下文
- Markdown 審查報告匯出

## 本版內容

### 審查工作流

- 行為一致性審查
- 對白風格審查
- 寫作風格審查
- 未手動選角時，自動從章節文字辨識角色

### 故事專案結構

- Markdown 角色設定檔
- Markdown 世界觀設定檔
- Markdown 寫作風格設定檔
- 關係、時間軸、伏筆的本地 JSON 持久化

### 專案基礎設施

- MIT 授權
- 格式、測試與 build 的 CI workflow
- 貢獻指南、安全政策與行為準則
- issue templates 與 pull request template
- 範例章節與架構文件

## 備註

- 這是一個早期但已可使用的本地工作流版本。
- 使用前需確認 Ollama 已在本地執行，且對應模型已安裝。
- `data/` 下的本地狀態檔維持不納入版控。

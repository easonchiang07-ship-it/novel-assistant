# Security Policy

## Supported Versions

Novel Assistant is currently in active early development. Security fixes are applied to the latest mainline version only.

## Reporting a Vulnerability

Please do not open public issues for security-sensitive problems.

Report through one of these paths:

- GitHub Security Advisories, if enabled for the repository
- A private maintainer contact through GitHub

Please include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested mitigation

We will review reports as quickly as possible and coordinate disclosure after a fix is ready.

## Security Notes

- This project is designed for local-first use.
- Review carefully before committing story data or private manuscripts to GitHub.
- Ollama runs locally, but users should still verify model names, ports, and machine access policies in their own environment.
